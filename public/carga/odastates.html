Vistas de odas Recrea

  • OBTENER DATOS DE ESTUDIANTE AL FINALIZAR
    Vista normal. Al finalizar se reciben los datos del estudiante.
  • ASIGNAR NÚMERO DE INTENTOS
    Se manda el número de intentos que lleva el estudiante.
  • CARGAR DATOS DE ESTUDIANTE
    Carga los datos previamente ingresados por el estudiante. Los datos pueden ser modificados y de nuevo guardados con "Finalize"
  • CARGAR DATOS DE ESTUDIANTE : CONGELADO ❄️
    Carga los datos previamente ingresados por el estudiante. Es únicamente de consulta. El botón "finalize" NO envía datos para guardar. Cuando los datos del estudiante estan congelados se vuelve morado el contorno de la oda.
  • VISTA DE MAESTRO (+ CARGAR DATOS DE ESTUDIANTE: CONGELADO ❄️)
    La vista del maestro requiere que también se carguen los datos del estudiante. Se presenta el botón "Save evaluation and comments" que manda los datos ingresados por el maestro para ser guardados.
  • CARGAR DATOS DE MAESTRO (+ CARGAR DATOS DE ESTUDIANTE: CONGELADO ❄️ + VISTA DE MAESTRO)
    Se puede cargar la información previamente ingresada por el maestro para modificarla. Al presionar el botón (Save evaluation and comments) se vuelven a enviar los datos. Opcional*
  • CARGAR DATOS DE MAESTRO : CONGELADO (+ CARGAR DATOS DE ESTUDIANTE: CONGELADO ❄️ + VISTA DE MAESTRO)
    Sería la vista que se le presenta al estudiante una vez que fue evaluado por el maestro. Se congelan los datos del usuario y los del maestro.
